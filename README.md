# Java Fighter Game

## Description
This is a console-based fighting game developed in Java using Object-Oriented Programming principles.

## Features
- Create fighters (Knight, Wizard, Assassin)
- Each fighter has unique abilities
- Battle system
- Console-based interaction

## Technologies
- Java
- OOP (Inheritance, Polymorphism)

## How to Run
1. Compile the Java files
2. Run the Main class
